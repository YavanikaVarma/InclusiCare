# InclusiCare - Your Safe Space

InclusiCare is a privacy-first, web-based mental wellness platform built to provide emotional safety, anonymity, and calm healing experiences.

## Features
- **Empathetic AI Companion**: Friend & Crisis modes.
- **Panic Button**: Instant 4-7-8 breathing support.
- **Smart Journaling**: Private, local storage based.
- **Wellbeing Meter**: Mood tracking.
- **Privacy First**: No data collection, all local.

## How to Run locally

### Prerequisites
- Node.js installed.

### Steps
1. Open this folder in your terminal.
2. Initialize npm (if not already done):
   ```bash
   npm init -y
   npm install express
   ```
3. Start the server:
   ```bash
   node server.js
   ```
4. Open your browser and go to `http://localhost:3000`.
